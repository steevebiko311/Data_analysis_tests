# Data_analysis_tests
Pratical group work

Voici les étapes pour la méthode sans collaboration (fork et pull request) :

*Étapes pour les membres du groupe :*

1. *Créer un fork du repository* :
- Aller sur le repository sur GitHub
- Cliquer sur le bouton "Fork" en haut à droite
- Cela créera une copie du repository sur votre compte GitHub
2. *Cloner le fork* :
- Ouvrir un terminal et taper : `git clone <lien_du_fork>`
- Cela créera une copie locale du repository sur votre ordinateur
3. *Faire vos modifications* :
- Ajouter, modifier, supprimer des fichiers dans le repository local
4. *Ajouter les modifications* :
- `git add .` (ajoute tous les fichiers modifiés)
- ou `git add <fichier>` (ajoute un fichier spécifique)
5. *Committer les modifications* :
- `git commit -m "Message de commit"` (remplacer "Message de commit" par un message descriptif)
6. *Pousser les modifications* :
- `git push origin <branche>` (remplacer "<branche>" par la branche que vous avez créée)
7. *Créer une pull request* :
- Aller sur le repository original et cliquer sur "New pull request"
- Sélectionner le fork et la branche que vous avez créée
- Décrire les modifications et soumettre la pull request
